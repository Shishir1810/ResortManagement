# ResortManagement
## Installation

Cloneing the repository Please Run These Commands:

composer update
copy  .env.example .env
php artisan key:generate

Then Configure the .env file

After Configuring


php artisan migrate:fresh --seed
php artisan optimize
php artisan serve

 admin will be created.
